# amazon_eda.py
End-to-end EDA on Amazon product sales data — cleaning, feature engineering, and uni/bi/multivariate analysis using Pandas, Matplotlib, and Seaborn.

Key insights explored

→ Do sponsored products sell more than non-sponsored?
→ Which product categories drive the most monthly purchases?
→ What is the correlation between product rating and purchases?
→ Revenue breakdown by category and sponsorship status (pivot table)
